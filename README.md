Fernus ile yayınlanmış video çözümleri tek bir çatı altında toplayan, kolay bir kullanım sunan internet sitesi.

<hr>

Next.js ve Tailwind CSS kullanılarak yapıldı. Şu anda geliştirme aşamasında.
